# ⚔️ 兵將王傳奇 (Legend of Kings and Generals)

**💡 核心目標：打敗敵方國王，並保護己方國王！**

《兵將王傳奇》是一款基於 Python 與 Tkinter 開發的 2D 網格戰略棋盤遊戲。玩家必須在有限的回合與行動點數（AP）內，利用不同的兵種與地形優勢，擊敗對手的國王以取得勝利，同時還需應對地圖上的第三方勢力（盜賊與叛軍）。

---

## ✨ 遊戲特色

* **多樣化的遊戲模式**：支援 PvP (玩家對戰)、PvAI (玩家對電腦)、AIvP (電腦對玩家) 以及 AI vs AI (純觀賞模式)。
* **豐富的地形系統**：包含平原、河流、橋樑、城堡、高山、森林與叛軍營地。不同地形會影響單位的移動距離與疲勞度（例如：渡河會消耗更多體力，甚至導致單位溺水）。
* **AP 行動點數機制**：每回合擁有 18 點 AP。玩家可以自由分配 AP 讓單位進行移動或攻擊。
* **連擊模式 (Streak Mode)**：開啟後，單位擊殺敵人後若還有剩餘體力與 AP，可消耗額外 AP 進行連續攻擊。
* **第三方勢力干擾**：除了紅藍兩軍，地圖上還會生成中立盜賊與叛軍，他們會在專屬回合自動行動並攻擊鄰近目標。
* **現代化 UI 與操作**：支援右鍵拖曳地圖、滾輪縮放畫面，並附帶即時小地圖與悔棋功能。
* **打包相容**：內建資源路徑轉換，完美支援 PyInstaller 打包成獨立執行檔 (EXE)。

## 🎮 玩法說明

### 兵種與分數 (Score)
* **王 (King)**: 價值 10 分。核心單位，國王死亡即宣告戰敗。（第 7 回合前限制在城堡範圍內）
* **將 (General)**: 價值 5 分。戰鬥力強，具有保護周圍單位的靈氣效果，剛晉升時具有短暫無敵狀態。
* **兵 (Pawn)**: 價值 2 分。基礎單位，擊殺敵人後有機會晉升為「將」。
* **賊 (Thief) / 叛 (Rebel)**: 第三方勢力單位。

### 基本操作
* **左鍵點擊**：選取己方單位 / 點擊亮起的地圖格子進行移動或攻擊。
* **右鍵拖曳**：平移遊戲地圖。
* **滑鼠滾輪**：放大或縮小地圖。
* **小地圖點擊/拖曳**：快速跳轉視野。
